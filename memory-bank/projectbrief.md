# AlgoAgentX Project Brief

## Project Overview
AlgoAgentX is a comprehensive SaaS platform for algorithmic trading, featuring:
- **Frontend**: Next.js App Router with internationalization (i18n) support
- **Backend**: FastAPI with SQLAlchemy, PostgreSQL database
- **Architecture**: Full-stack SaaS with user authentication, role-based access, credit system, and payment integration

## Core Requirements
1. **User Management**: Authentication, profiles, role-based access (user/admin)
2. **Trading Tools**: AI Screener, Backtesting engine, Strategy management
3. **Payment System**: Razorpay integration with credit-based billing
4. **Admin Dashboard**: Complete admin interface for managing users, payments, orders, support
5. **Production Ready**: Professional UI/UX, error handling, security, performance

## Current State
- Basic authentication system implemented
- Database models for users, strategies, backtests, notifications exist
- Frontend has basic dashboard and auth pages
- Need to implement full admin system and polish existing features

## Target Outcome
Professional SaaS platform ready for production deployment with:
- Complete feature set (user + admin interfaces)
- Razorpay payment integration
- Credit-based billing system
- Professional UI with dark mode, notifications, etc.
- Admin dashboard for complete system management
